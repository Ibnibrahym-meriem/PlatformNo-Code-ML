# No-Code ML Platform 🚀

## 📌 Description
**No-Code ML Platform** est une application web interactive permettant aux utilisateurs de créer, prétraiter, analyser et visualiser des données sans écrire de code. Elle propose des dashboards dynamiques, des outils de prétraitement, des visualisations avancées et une interface moderne orientée expérience utilisateur.

## 🎯 Objectifs
- **Faciliter l’analyse de données** sans compétences techniques avancées
- **Offrir des dashboards interactifs** et personnalisables
- **Centraliser** les étapes de prétraitement, d’analyse et de visualisation
- **Améliorer la prise de décision** grâce à des indicateurs clairs (KPIs)

## 🧩 Fonctionnalités principales
- 📊 **Dashboards interactifs** avec KPIs dynamiques
- 🧹 **Prétraitement des données** (nettoyage, normalisation, encodage, etc.)
- 📈 **Visualisations** (charts, statistiques descriptives, aperçu des données)
- 📄 **Pages multiples** pour une navigation fluide
- 🎨 **Design moderne et responsive**
- ⚙️ **Architecture No-Code** orientée utilisateur

## 🛠️ Technologies utilisées

### Frontend
- **React.js**
- **Vite**
- **Tailwind CSS**
- **Recharts**

### Backend 
- **FastAPI**
- **Python**
- **Scikit-learn / Pandas**
